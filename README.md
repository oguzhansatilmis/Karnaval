# Getting Started

This project consists of 2 screens. Movies are listed and details of the selected movie can be accessed.

## Used technologies:

- Kotlin
- Android Jetpack (Navigation, ViewModel, Livedata, Room, Lifecycle),
- MVVM,Clean Code
- Coroutines
- Retrofit
- Dagger & Hilt
- Unit Test
